# ADR 0001: Android, wired-first

## Status
Accepted.

## Decision
Build the user-facing prototype on Android, but support experimental cancellation only through wired analog or USB-C audio routes initially.

## Rationale
Android reaches the target audience, while wired routes provide lower and more stable latency than ordinary Bluetooth paths.

## Consequences
Bluetooth may be used for analysis experiments but is not part of initial cancellation claims.
